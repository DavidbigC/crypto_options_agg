# Polysis Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a standalone `/polysis` page that turns Polymarket crypto prediction markets into a volatility-analysis dashboard and compares the derived signal against the existing options analysis surface.

**Architecture:** Add a backend Polymarket adapter that discovers, prices, filters, and normalizes related prediction markets into one asset-and-horizon payload. Expose a compact backend API for the frontend page, then render a chart-led dashboard with quality scoring, repricing, and options comparison panels.

**Tech Stack:** Node.js, Express, Next.js App Router, React, TypeScript, existing frontend charting stack, backend fetch adapters, existing analysis helpers where reusable
